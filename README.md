## üéØ How This Works

**No rigid rules** ‚Äî Update when it makes sense, not on a schedule.

**Categories flow naturally:**

* If you're actively doing it and it's generating results ‚Üí **Live & Running**
* If you're building something new ‚Üí **Building**
* If it's creative work that's always happening ‚Üí **Ongoing Practice**
* If it's home/life stuff ‚Üí **Home & Life Projects**
* If it's a good idea for later ‚Üí **Someday/Maybe**
* If it's done ‚Üí **Complete**

**Weekend Tasks** ‚Äî Small jobs (under 4 hours) live in Quick Tasks, not projects.

**When to review:** When things feel cluttered, when priorities shift, or when you've got new momentum and want to capture it.
